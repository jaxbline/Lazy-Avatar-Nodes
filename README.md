# Requirements
* Blender 4.0 +


# Installation/Usage
* [Goto releases](https://github.com/jaxbline/PoiyomiNodes/releases)
* Download latest
* Open Blender
* Goto Edit -> Preferences -> Add-ons
* Install Lazy Avatar Nodes
